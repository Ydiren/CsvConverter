# CsvConverter

```
  Usage: CsvConverter [options] <Input> <Output>

  Arguments:
    Input
    Output

  Options:
    -i|--input-type <INPUT_TYPE>    InputType
    -o|--output-type <OUTPUT_TYPE>  OutputType
    -?|-h|--help                    Show help information.

    Accepted Input Types:
        csv

    Accepted Output Types:
        xml, json
```
